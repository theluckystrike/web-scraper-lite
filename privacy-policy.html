Privacy Policy - Web Scraper Lite

Last Updated: February 2026

Developer: Web Scraper Lite Team

Extension Name: Web Scraper Lite

Extension ID: web_scraper_lite

Overview

Web Scraper Lite is a Chrome extension that enables users to extract structured data from web pages using a visual point-and-click interface. This privacy policy explains how we handle your data and complies with Chrome Web Store requirements.

Data Collection Summary

Web Scraper Lite does not collect any personal data for our servers.

We do not collect:

  • Your name, email, or contact information (except when you provide a license key)
  • The content you scrape from websites
  • Your browsing history or activity
  • Device identifiers or fingerprints
  • Location data
Important: All analytics and usage data is stored locally on your device and is never transmitted to external servers (except for license verification with Zovo).

Data Transmission

Almost all data stays on your device. The only exception is license verification for Pro users, detailed below.

For Free users:

  • No analytics are sent anywhere
  • No usage data is uploaded
  • No scraped content leaves your browser
  • No crash reports are transmitted
  • The extension works fully offline after installation

Third-Party Services

This extension does not use:

  • Analytics services (Google Analytics, Mixpanel, etc.)
  • Advertising networks
  • Cloud storage services
  • Crash reporting services (Sentry, etc.)

License Verification (Zovo)

Web Scraper Lite Pro licenses are managed through Zovo, a unified payment system for browser extensions. When you have a Pro license, the extension periodically verifies your subscription status.

Data sent to Zovo:

Data Purpose
License key Verify your subscription is active
Extension ID (web_scraper_lite) Identify which product to verify
Session ID Anonymous session tracking for fraud prevention

Data NOT sent to Zovo:

  • Your scraped content or extracted data
  • URLs you visit or scrape
  • Your browsing history
  • Personal files on your computer
  • CSS selectors or template configurations
  • Any locally stored analytics or usage data

Verification frequency:

  • When you open the extension
  • Approximately every hour while the extension is active
  • Results are cached locally for 5 minutes to minimize API calls

API endpoint: https://xggdjlurppfcytxqoozs.supabase.co/functions/v1

See Zovo's privacy policy for details on how they handle license data.

Permissions Explained

Permission Why It's Needed
activeTab Allows the extension to interact with the current tab when you click the extension icon. Without this, we cannot highlight elements or extract data from the page you're viewing.
storage Enables saving your templates, preferences, and scrape history locally on your device. This data never leaves your browser.
scripting Required to inject the element selector overlay onto web pages. This is what highlights elements when you hover over them and captures your clicks.
sidePanel Powers the side panel interface where you configure scrapers and preview extracted data.
Host permissions (all URLs) Allows the extension to work on any website you choose to scrape. Without this, the extension would only work on pre-approved sites.

Data Extracted from Websites

When you scrape a website, the extracted data is:

  • Displayed in the extension's preview panel within your browser
  • Exported to your local device as CSV or JSON files when you click Export
  • Never transmitted to our servers or any third party

The extension extracts only the elements you specifically select. It does not:

  • Capture passwords or form inputs
  • Access data from other tabs
  • Run in the background when not in use
  • Store complete page copies
Your Responsibility: You are responsible for complying with the terms of service and robots.txt of websites you scrape. Some websites prohibit automated data extraction. Web Scraper Lite is a tool; how you use it is your responsibility.

Your Rights

Because all your data is stored locally, you have complete control:

  • Access: View all stored data in Chrome's developer tools (Application > Local Storage > chrome-extension://[extension-id])
  • Delete: Clear all extension data by:
    1. Right-clicking the extension icon > "Remove from Chrome"
    2. Or: Chrome Settings > Extensions > Web Scraper Lite > Details > Clear data
    3. Or: Developer Tools > Application > Storage > Clear site data
  • Export: Export your templates before uninstalling if you want to preserve them
  • Portability: Your local data stays on your device and can be backed up
  • Opt-out of analytics: All analytics are local-only, but you can clear them anytime via Chrome's storage settings

We cannot access, retrieve, or delete your data because we never have it in the first place. The only exception is license data stored with Zovo - contact Zovo support to delete your license information from their systems.

Children's Privacy

This extension is not directed at children under 13 years of age. We do not knowingly collect any information from children. The extension has no accounts, no data collection, and no way to identify users of any age.

Security

  • The extension uses Chrome's Manifest V3, which enforces stricter security policies
  • Content Security Policy prevents execution of unauthorized code
  • No external scripts are loaded
  • All code is reviewable in our public GitHub repository
